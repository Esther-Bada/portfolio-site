01 / Problem

No setup,
no signal.

  • ✗ Notepad as an editor — no syntax highlighting, no formatting, no feedback
  • ✗ Manual browser refreshes after every single save
  • ✗ No version control visibility — git felt abstract and forgettable
  • ✗ When something broke, no way to isolate why — just trial and error

"I spent an hour staring at a CSS bug. A better-equipped developer fixed it in 25 seconds. The difference wasn't skill. It was tools."

— The moment it became real
02 / Method

One pain
at a time.

Editor
VS Code + Live Server
Installed when manual refreshing became unbearable. One button — instant browser preview on every save.
Formatting
Prettier
Added when my own code started confusing me. Enforces consistency automatically — no decisions needed.
Version control
GitLens
Made git history visible inline. Turned an abstract habit into a visible story of the work.
AI pair programming
Claude Code
Reads your actual files, edits code directly, explains what it changed. Used when hitting a real wall.
Principle: Don't install for completeness — install for pain. Each tool earns its place by solving a specific friction you've already felt.
